AI MARIO

an AI learns to play mario.

1. learning method

ga — best brains are kept, crossed and mutated; learns only from the final score.

neat — the wiring evolves too: minimal brains grow neurons & connections.

dqn — one brain learns while playing; ε = how randomly it acts.

2. ai inputs

what the AI can sense. more inputs = more information but a bigger, slower network — a few well-chosen senses learn fastest.

tile grid = tiles in front, marked solid / hazard / coin — the richest input.

changing inputs resets all networks — the input layer size changes.

3. ai outputs

the network picks one action each frame; fewer = faster learning. changing them resets all networks.

4. network

the size of the brain. more layers / neurons can express smarter behavior, but train slower and may overfit the training maps. for neat the topology is not yours to set — it grows nodes & connections by mutation, starting from inputs wired straight to outputs.

the picture shows the current best brain: green = positive weight, red = negative; node color = activation while watching.

5. training

rewards shape personality: raise coin and it detours for coins; punish jumps and it hops only when needed. all three methods optimize this same score (the dqn receives it per step). distance is scaled by progress (1000 → 1% of the level = 10 points) · time is charged per second.

fit last X maps: a fresh random level is added every generation/iteration and scores are averaged over the last X, forcing it to generalize instead of memorizing.

the species compatibility threshold self-adjusts to hold roughly the target number of species.

episodes rotate through the last X maps while the agent learns from per-step rewards with ε-greedy exploration (ε shown in the stats decays as it learns).

train runs until you stop it · watch best replays the best brain with its senses drawn · new level injects an unseen map.

hazards: pit / spike / enemy = death (goombas & flyers can be stomped, red spinies cannot) · springs bounce high · reach the flag to win.

6. progress

▬ black = best score · ▬ grey = population / episode average · ▬ blue = benchmark: the best brain on a hidden level it never trains on, per generation / iteration — each method keeps its own history.

if black climbs but blue doesn’t, it is memorizing, not learning.
